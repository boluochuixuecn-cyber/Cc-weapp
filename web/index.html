CC UI - 网页版示例与文档

本页面为静态示例，帮助快速浏览组件 API 与代码片段。真实效果请在「微信开发者工具」中打开 pages/index/index 进行预览。

快速开始

在页面 index.json 引入需要的组件：

{
  "usingComponents": {
    "cc-popup": "/components/cc-ui/popup/popup",
    "cc-button": "/components/cc-ui/button/button",
    "cc-toast": "/components/cc-ui/toast/toast",
    "cc-dialog": "/components/cc-ui/dialog/dialog",
    "cc-icon": "/components/cc-ui/icon/icon"
  }
}

Popup 弹出层

弹出层容器，用于展示弹窗、信息提示等内容。

<cc-button type="primary" bind:tap="showPopup">展示弹出层</cc-button>
<cc-popup show="{{ show }}" bind:close="onClose">内容</cc-popup>

Button 按钮

支持多尺寸、朴素、块级、圆角、加载等。

<cc-button type="primary">主要按钮</cc-button>

Toast 轻提示

用于轻量级操作反馈与状态提示。

// wxml
<cc-toast id="ccToast" />
// js
const toast = this.selectComponent('#ccToast');
toast.showToast({ message: '操作成功' });
查看 API

Dialog 对话框

模态确认框，支持异步关闭。

// wxml
<cc-dialog id="ccDialog" bind:confirm="onConfirm" bind:cancel="onCancel" />
// js
const dialog = this.selectComponent('#ccDialog');
dialog.open({ title: '提示', message: '确认执行该操作？' });
